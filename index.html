Шифр Виженера

Метод полиалфавитного шифроавания буквенного текста с использованием ключевого слова. Этот метод является простой форма многоалфавитной замены.

Принцип работы

В качестве ключа выступает специальная фраза. Она несколько раз повторяется и пишется над шифруемым текстом. В результате каждая буква посылаемого сообщения сдвигается относительно заданного текста на определенное число, которое задается буквой ключевой фразы. На протяжении нескольких веков данный метод устойчиво занимал позицию самого надежного метода шифрования.

Принцип работы программы

Данная программа шифрует помимо букв и специальные символы, включая пробелы для сохранения целостности информации. Кроме того для упрощения работы с текстом в программе есть функция чтения файлов .txt расширения. Данная программа работает с раскладкой ACLL, что означает её не поддержку русского языка на данный момент. Кроме того программа может дешифрировать сообщения при знании правильного ключа.
